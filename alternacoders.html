Creating a .bash_profile

So this is a little intimidating, you are basically playing with the brain of your computer, but moreso you are preparing your tool set so that your work will be more sophisticated and informed. Also, as long as you pay attention, you will probably not kill your patient.

cd ~/

~touch .bash_profile / Ubuntu uses ~/.profile This creates a new file

or ~ open ~/.bash_profile Which automatically opens it in TextEdit.

You can also use nano .bash_profile

^= Control

source .bash_profile shows what changes have been made

Mine looks like this:

alias ll='ls -ltrahG'

[[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm" #Load RVM into a shell session *as a function*

export PATH="$HOME/.rbenv/bin:$PATH"

# indicates commenting. I like commenting, it makes for a much better experience later, even if it is only me using it. I use Ruby mostly, so that's why I have a ruby specific profile.

Saving Your bash profile across computers

Create a file in Dropbox, something like: ~/Dropbox/Apps/Terminal/profile.txt

Then enter ~ln -s ~/Dropbox/Apps/Terminal/profile.txt ~/.profile
