cd ~/

~touch .bash_profile / Ubuntu uses ~/.profile This creates a new file

or ~ open ~/.bash_profile Which automatically opens it in TextEdit.

Mine looks like this:

alias ll='ls -ltrahG'

export PATH="$HOME/.rbenv/bin:$PATH"

# indicates commenting. I like commenting, it makes for a much better experience later, even if it is only me using it. I use Ruby mostly, so that's why I have a ruby specific profile.
